# Product Context

## Project Purpose
This appears to be a Smart Farm monitoring system using ESP32 hardware for agricultural monitoring and control.

## Core User Problems/Solutions
- Need for remote monitoring of agricultural conditions
- Automation of farm monitoring processes
- Data collection and visualization for agricultural metrics

## Key Workflows
1. Sensor data collection
2. Data visualization (using OLED display)
3. Remote monitoring (via MQTT)
4. Web interface (using ESPAsyncWebServer)

## Product Priorities
1. Reliable sensor data collection and processing
2. Real-time monitoring capabilities
3. User-friendly interface (both physical OLED and web interface)
4. Stable connectivity for remote access

## Integration Points
- MQTT broker for remote monitoring
- Web interface for local access
- OLED display for on-device information
- Various sensors for data collection
